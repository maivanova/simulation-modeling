# Математическая модель

Рассматривалась модель экспоненциального роста:

$$
u(t) = u_0 e^{\alpha t}
$$

где:
- $u_0 = 1.0$
- $\alpha = 0.3$

---

# Реализация

- на языке Julia вычислены значения функции на интервале от 0 до 10;
- результаты сохранены в CSV-файл;
- по CSV построен график;
- подготовлена literate-версия программы.
