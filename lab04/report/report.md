# Лабораторная работа №4

Реализовать агентную модель SIR на графе, выполнить базовый эксперимент и параметрическое сканирование коэффициента заразности.

## Выполнение работы

Был создан проект `lab04` с агентной моделью SIR.  
Выполнены:
- базовый эксперимент;
- построение графика динамики S, I, R;
- параметрический эксперимент по коэффициенту β;
- подготовка literate-файлов;
- генерация `.jl`, `.ipynb`, `.qmd`.

## Результаты

Получены:

- `beta_scan.png`;
- literate-артефакты;
- notebook и Quarto-документация.
